If you do not want to use pre-compiled CSS files in your project you can bundle and build Oslo style guide yourself for more fine grained control.

Install
Add Oslo style guide to your project using for example npm: npm install git://github.com/oslokommune/styleguide.git#1.0. Tune versioning according to semver.org. Available versions can be found at https://github.com/oslokommune/styleguide/releases.

Build
You need a build tool to compile your stylesheets. It needs to support the sass-dart compiler.

SASS

If you chose NPM / SASS you can either @use "~styleguide/src/osg.scss" for the complete package or individually include the components you want.
