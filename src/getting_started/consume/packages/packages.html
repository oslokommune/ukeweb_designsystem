If you do not want to bundle and build Oslo style guide in your project, you can use an already ready to use package. Each version of Oslo style guide is built and available for direct consumption.

Complete
Includes a reset, some basic styles, fonts, icons and everything in General and Components. https://ukeweb-styleguide-cdn.s3.eu-central-1.amazonaws.com/1.0.0/osg.css

Versions
https://ukeweb-styleguide-cdn.s3.eu-central-1.amazonaws.com/[VERSION]/osg.css

See https://github.com/oslokommune/styleguide/releases for available versions and changelogs.
