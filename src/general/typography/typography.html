Style

Italic

Use em or .ods-text-style-italic class if not possible

Strong

Use strong or .ods-text--weight-medium class if not possible

Text modifiers

Nowrap

Text modifiers are supported for all classes. For example:

.ods-text--modifier-nowrap

.ods-text--modifier-nowrap-breakpoint-medium

.ods-text--modifier-nowrap-breakpoint-large

nowrap example: lorem ipsum dolor sit amet consectetur adipiscing elit sed do eiusmod tempor incididunt ut labore et dolore magna aliqua ut enim ad minim veniam quis nostrud

Truncated

.ods-text--modifier-truncated

.ods-text--modifier-truncated-breakpoint-medium

.ods-text--modifier-truncated-breakpoint-large

Be careful when truncating text as the accessibility is limited. Truncation may cause issues with accessibility tools like screen readers.

truncated example: lorem ipsum dolor sit amet consectetur adipiscing elit sed do eiusmod tempor incididunt ut labore et dolore magna aliqua ut enim ad minim veniam quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat

Breakpoints

Breakpoint modifiers are supported for all classes. For example:

.ods-text--size-kilo-breakpoint-medium

.ods-text--size-delta-breakpoint-large

.ods-text--weight-medium-breakpoint-medium

.ods-text--style-italic-breakpoint-large

.ods-text--align-right-breakpoint-large
