A radio button is a graphical control element that allows the user to choose only one of a predefined set of mutually exclusive options.

Rules for components

The radio button text should never take the 100% width of the page in medium and up breakpoints.
Use .osg-radio__label--align-top modifier to align the text to the top when it needs more than one line.
